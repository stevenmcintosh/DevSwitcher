Updates

v2.1

  • Completely re-written the meta checking functions to increase performance
  • The meta tag checking no longer takes place on non-pokerstars domains or pokerstars.net domains.

v2.0

  • Complete overall of code (except meta checking section), boosting both performance and enhancements to existing features.
  • New foundations so that new functions can be added easier
  • New toggle net/com button to help easily toggle between com and net domains

v1.4

  • New function where the DOM is checked for valid meta title and description. The number of chars (colour coded for validity) for both metas are displayed to help with quick SEO checks.

v1.3

  • The url of the broken canonical is now the tooptip of the error msg in the toolbar
  • If event is clicked and not on PokerStars domain, then jump straight to the clicked event on pokerstars.com
  • Added .jp (Japanese) to the list of langs
  • Added .tw (Taiwanese) to the list of langs
  • Added debug mode to help switch debugging on/off easily
  • Added quick launch events where use can quickly jump to any pokerstars event/tournament.
  • New user preferences panel 'events' where user can choose which events to show on toolbar.
  • Fixed bug where .net domains to .com domains sometimes shown undefined in url

v1.2

  • If a PokerStars page is loaded that has no canonical, a canonical error is shown on the toolbar
  • If not on a PokerStars domain and you click a language on the toolbar, you will go a straight to the PokerStars homepage of te clicked language
  • Current language is highlighted red
  • Switching languages whilst on .NET domains is now possible

v1.1

  • New View Source function, that when clicked toggles the current page source code.
  • New clone current tab function, that when clicked opens a new tab with the same url
  • Added a new toolbar palette icon which (spade with a star) that when clicked opens www.pokerstars.com
  • Added .co.uk to the list of langs
  • Removed all live debugging code
  • Updated options panel to work in Mac OS X

v1.0

  • When your not on a PokerStars website, you can now click on Dev, QA or WWW and will be taken to the respective website
  • Added 'Current Promotions' to the popular page quick links
    www.pokerstars.com/poker/promotions/
  • Added 'Real Money' to the popular page quick links
    www.pokerstars.com/poker/real-money/
  • Added .co.uk to the list of langs
  • Removed all live debugging code
  • Updated options panel to work in Mac OS X

v0.23

  • Bug Squashed: Fixed bug where if multiple instances of the same language appeared in url it splits url too many times.
    ie. www.pokerstars.pl/poker/promotions/ceepl/

v0.01 - v0.22

  • Switch easily between several development environments
    ie. dev.pokerstars.com → www.pokerstars.com
  • Switch easily between pokerstars language sites
    ie. www.pokerstars.com → www.pokerstars.cz
  • Switch easily to the popular pages
    ie. www.pokerstars.com → www.pokerstars.com/poker/promotions/
  • Switch easily from temp files to a development url
    ie. file:///X:/pokerstars.si/htdocs/wcp/ → www.pokerstars.si/wcp/
  • Ability to add custom buttons via the toolbar palette (If you dont want the full toolbar)
  • New user options dialog so that users can set their toolbar preferences
