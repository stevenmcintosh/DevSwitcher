A) HTML Files & Structure - top

Basics

All pages have been splitted in to three "main" sections; #header, #content and #footer. Inside each of these divs is wrapper div called .container, which determine the width of page and also centralize it. So just by adjusting .container width value you can either increase or decrease the width of whole layout.

Here is simplified example of page structure:

<div id="header">

    <div class="container">
	
        [HEADER_CONTENT_HERE]
        
    </div><!-- end .container -->
    
</div><!-- end #header -->

<div class="clear"></div>

<div id="content">

    <div class="container">
	
        [CONTENT_HERE]

    </div><!-- end .container -->

</div><!-- end #content -->

<div id="footer">

    <div class="container">
	
        [FOOTER_CONTENT_HERE]
		
    </div><!-- end .container -->
    
</div><!-- end #footer -->

You can also split the content area into #main and #sidebar, which quite obviously allow you to use sidebar:

<div id="content">

    <div class="container">
	
        <div id="main">
	
            [MAIN_CONTENT_HERE]

        </div><!-- end #main -->
	
        <div id="sidebar">
	
            [SIDEBAR_CONTENT_HERE]

        </div><!-- end #sidebar -->

    </div><!-- end .container -->

</div><!-- end #content -->

Customizing logo is extremely simple because it doesn't use any images! Of course you may need to change font-size and/or margin-top value if your site's name is really long. But as long you stay under 15 characters you don't need to change anything else than just a site name itself.

Socials links and navigation are just basic unordered lists. Class .active hightlights current page and to use submenu just create unordered list inside of top-level list item.

Home (index.html)

Before we start to explore projects grid system let's first check what else you can find from homepage. Homepage is splitted in different sections and every sections has its own header. By default you can only use two different icons but of course you can include you own ones too if you wish:

<div class="section-title">

    <h6><span class="icon pen"></span> [MY_TITLE] </h6>

</div><!-- end .section-title -->

You will also find carousel from bottom of the page and in fact it can hold any content you need. As long you make sure all content are placed inside of list items, it will allow you to use images, text and even videos.

Home » Projects

Now to the part that might be to reason why you bought this theme! This theme doesn't have any separate portfolio page but instead it all happens right in homepage, which is the most effective way to keep all visitors on your site. The projects grid itself it's placed in .projects-slider-gallery which is wrapped in #projects-slider.

You have five different device to choose from: browser, laptop, monitor, phone and tablet. If you don't need any device, just remove class attribute from span tag (but don't remove whole span tag!). You can also mark any item as new by adding this litte snippet just before anchor tag: <span class="new">New</span>.

Example of project:

<li>
    <span class="new">New</span>	
    <a data-project="my-project">
        <span class="phone">
            <img src="img/project-1.jpg" alt="Timy">
        </span>
        <h6>My Project</h6>
    </a>
</li>

The important part here is the data-project. As you can see above, anchor's data-project value is my-project and this value need to be exactly same in single project too. If these two doesn't match, it just won't work properly. Remember also that because this is static page, you need to change pagination values by hand.

With single project you have all same devices available but you have also one extra; phone-big. All single projects are placed in #single-project.

Simplified version of single project:

<div data-project="my-project">

    <div class="container">

        <h2>My Project</h2>

        <div class="controls">

            <ul class="pagination">
                <li class="prev"><span class="disabled">«</span></li>    
                <li><span>1 <em>of</em> 12</span></li>        
                <li class="next"><a href="#">»</a></li>                        
            </ul>

            <a href="#" class="exit">X</a>
            
        </div><!-- end .controls -->
            
        <div class="clear"></div>

        <ul class="slider">
            <li><span class="phone"><img src="img/project-single-1.jpg" alt="Image 1"></span></li>
            <li><span class="phone-big"><img src="img/project-single-2.jpg" alt="Image 1"></span></li>
        </ul>

        [ANY_OPTIONAL_CONTENT]
            
    </div><!-- end .container -->
	
</div><!-- end my-project (single project) -->

You can put anything you want in to [ANY_OPTIONAL_CONTENT]; text, images etc. The examples can be seen from source code.

Subpages

Markup for subpage is no different than in homepage but in subpages you can use breadcrumbs. It will be placed between #header and #content:

<div id="breadcrumbs">

    <div class="container">

        <ul class="breadcrumbs">
            <li class="home"><a href="index.html">Home</a></li>
            <li>&rarr;</li>
            <li>The Blog</li>
        </ul>
        
    </div><!-- end .container -->
    
</div><!-- end #breadcrumbs -->

Page header with search form:

<div class="page-header clearfix">

    <h1 class="page-title">Blog</h1>

    <div id="search-form" class="search">
        <input type="text" value="" placeholder="Search the blog" />
        <input type="submit" value="Go" />
    </div><!-- end #search-form -->
			
</div><!-- end .page-header -->

Subpages » Contact Us (contact.html)

Contact form configuration is really simple task. All you need to do is to open php/config.php file and change $your_email value to your own one and that's it!

<?php

// Email address where all messages are sent
$your_email = 'thisismy@emailaddress.com'; // Replace this with your own email address

C) JavaScript - top

All Javascript files are placed in js directory:

  1. custom.js - All Javascript settings & custom scripts
  2. jquery.gridnav.js - Portfolio items rotator
  3. jquery.cycle.all.js - A slideshow plugin that supports many different types of transition effects
  4. jquery.jcarousel.min.js - A jQuery plugin for controlling a list of items in horizontal or vertical order
  5. jquery.fancybox-1.3.4.pack.js - Fancy lightbox alternative
  6. jquery.easing.1.3.js - More transition effects
  7. gmap3.min.js - A jQuery plugin to use google maps version 3
  8. IE9.js - A JavaScript library to make Microsoft Internet Explorer behave like a standards-compliant browser
  9. selectivizr-min.js - CSS3 pseudo-class and attribute selectors for IE 6-8
