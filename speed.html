Proofs in the pudding

The magic number is 37. Why? Because we currently have 37 languages

If you work for PokerStars, the chances are this little gem will save you time. That should be all you need to know, however, if its proof or a little persuation your after, then here are a few examples to get your juices flowing.

Changing development environments

Without DevSwitcher

It takes approximately 5 seconds to change the URL each time you wish to view the file on a different development server

With DevSwitcher

It takes approximately 1 second to change the URL with DevSwitcher
